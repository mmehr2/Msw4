Hand Controller Installation

After plugging in the hand controller, you will be presented with a ‘New Hardware’ wizard. This wizard varies with different versions of Windows, but the concepts are similar.

1.                   Log in as administrator.

2.          If asked to use Windows Update or the internet to find a driver, choose 'No'.

3.                   If asked to install automatically or manually (specify a disk or location), choose manual.

4.                   When asked the location of the driver or software, browse to the folder where you have installed MagicScroll. If you have not yet installed MagicScroll, exit the wizard and install MagicScroll.

5.                   Ignore warnings that the driver is not digitally signed or from an ‘unknown provider.’

 

If installation fails or the hand controller does not work after completing the above procedure, usually the best course of action is to:

1.                   Open Device Manager by choosing Start | Control Panel | System, Hardware tab, and pressing the Device Manager button.

2.                   Find the entry for the controller (it will probably have a yellow question mark icon), right click on it, and choose ‘Uninstall.’

3.                   Now connect the controller to your computer again, and you should be presented with the ‘New Hardware’ wizard.
