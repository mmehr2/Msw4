Control Panel

The control panel is initially displayed across the top of the application window, below the menu bar and to the right of the toolbar.

The commands (from left to right) are:

  • Font
 Click this button to specify a size for the selected text. Shift+click allows you to choose font effects, paragraph justification, and single, 1.5, or double spacing. Alt+click will bring up the font dialog. Ctrl+click displays a menu of styles to choose from.
  • Find & Replace
 Invokes the Find & Replace dialog for searching for and replacing text.
  • Bookmarks
 Click to select a bookmark to jump to. Ctrl+click to clear all bookmarks. Alt+click to set a bookmark at the current cursor location.
  • Go Into Scroll Mode
 This command switches to scroll mode.
  • Timer to ART
 This is a push button that displays the current value of the ART (actual run time) timer. Pressing this button will place this value in the active (foreground) script's ART column on the Script Queue.
  • Loop
 When selected, the Loop command causes script(s) to repeat when scrolled. When the operator reaches the bottom of the script, the beginning of the script will again be displayed.
  • Out
  
  • Link
 When selected, the Link command causes multiple script(s) to be scrolled together. The scripts that will be scrolled can be viewed and specified in the Script Queue. If Link is not selected, the scroll window will only display the active (foreground) script or the first scrollable script as specified in the Script Queue.

The control panel can float or be moved to a different location by clicking and dragging on the grab handle to the left of the font button.

To hide or display the control panel, click Control Panel from the View menu.
