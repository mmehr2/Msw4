Cue Bar

The cue bar is displayed on the left side of the script window in edit mode.

The elements displayed on the cue bar are:

  • Line numbers
 These can be turned on an off using the Line Numbers command on the View menu
  • Paperclips
 A paperclip is displayed on its associated line of text.
  • Bookmarks
 A bookmark is displayed on its associated line of text.
  • Cue Pointer
 The cue pointer indicates the position where teleprompting should begin. It will be shown in the scroll window at the same vertical position. The pointer can be moved vertically by clicking and dragging anywhere on the pointer. Atl+click to choose a different or invisible cue pointer.

 

The cue bar can be included when printing your script by selecting the appropriate option in the Preferences dialog box.
