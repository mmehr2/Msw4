Edit Window Hot Keys

The following hot keys are available in edit mode:

Ctrl+N New
Ctrl+O Open
Ctrl+S Save
Ctrl+P Print
   
Ctrl+Z Undo
Ctrl+X Cut
Ctrl+C Copy
Ctrl+V Paste
Ctrl+A Select All
Ctrl+F / Ctrl+H Find and Replace
F3 Find Next
   
Ctrl+Q Show/Hide script queue
Ctrl+G / Esc Go into scroll mode
Ctrl+Alt+L Show/Hide line numbers
Ctrl+D NTSC display update
   
Ctrl+B Toggle bold
Ctrl+I Toggle italic
Ctrl+U Toggle underline
   
Ctrl+E Paragraph center align
Ctrl+R Paragraph right align
   
Alt+K Set bookmark
Ctrl+K Next bookmark
Ctrl+Shift+K Previous bookmark
Alt+P Toggle paperclip
Ctrl+Space Next paperclip
Ctrl+Shift+Space Previous paperclip
   
Ctrl+T Timer to ART
Ctrl+Shift+T Start timer in scroll mode
Ctrl+Alt+T Reset timer
Ctrl+L Goto line
   
Alt+Down arrow Scroll line down
Alt+Up arrow Scroll line up
   
F1 Help
