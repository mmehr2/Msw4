Format Bar

The format bar is initially displayed across the top of the application window, below the tool bar and control panel.

The controls and commands (from left to right) are:

  • Style
 Apply a style to the selected text by clicking on the drop-down arrow, then selecting the style. Styles can be added, removed, and modified by choosing Format | Styles from the main menu.
  • Font
 Apply a font to the selected text by clicking on the drop-down arrow, then selecting the font.
  • Font size
 Select a font size by typing directly in this control or clicking the drop-down arrow to choose a size.
  • Increase size
 Increases the size of the selected text or the entire document, if no text is selected.
  • Decrease size
 Decreases the size of the selected text or the entire document, if no text is selected.
  • Bold
 Toggles the bold state of the selected text.
  • Italics
 Toggles the italic state of the selected text.
  • Underline
 Toggles the underlined state of the selected text.
  • Color
 Displays a list of available colors to apply to the selected text.
  • Left align
 Left justifies the selected paragraphs or the paragraph containing the cursor, if there is no selection.
  • Center align

Center justifies the selected paragraphs or the paragraph containing the cursor, if there is no selection.
  • Right align

Right justifies the selected paragraphs or the paragraph containing the cursor, if there is no selection.
  • Bullet
  • Toggles bullet style for the selected text

The format bar can float or be moved to a different location by clicking and dragging on the grab handle to the left of the style box.

To hide or display the format bar, click Format Bar from the View menu.
