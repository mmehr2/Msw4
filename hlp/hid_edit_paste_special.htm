Paste Special command (Edit menu)

Use this command to insert a copy of the clipboard contents at the insertion point. A dialog presents you with various options. Select the format desired in the As: box. Below are some of the options you might see:

  • Unformatted Text will import the text without font, size, or color information.
  • Rich Text (RTF) will import the text with font, size, color, and other formatting information.

Warning: other formats such as Word Document, Windows Metafile, etc. will import embedded objects that will be displayed for printing purposes, but will not be available when scrolling.

This command is unavailable if the clipboard is empty.
