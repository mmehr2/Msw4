To create, open, or save a MagicScroll script

You can create, open, and save MagicScroll scripts by clicking the File menu, and then clicking New, Open, or Save.

Create a new document

On the File menu, click New, then begin typing.

Open a document

On the File menu, click Open. In Look in, click the drive and folder that contains the document you want to open. Locate the document, click it, and then click Open. If you do not see the document you want, you can click a different file type in Files of type. For example, click Unicode Text Documents to see only Unicode files on the list. Or you can open a document you opened recently by clicking its name on the File menu.

Save changes to a document

On the File menu, click Save. To save the file with a new name, on the File menu, click Save As, type a different name in File name, and then click Save. Documents that use multiple languages, formatting, bookmarks, or paperclips should be saved as rich text files.
