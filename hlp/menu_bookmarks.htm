Bookmarks menu commands

The Bookmarks menu offers the following commands:

Set Bookmark Creates a new bookmarks at the cursor in the current script.
Next Bookmark Jumps to the next bookmark in the current script.
Previous Bookmark Jumps to the previous bookmark in the current script.
Clear All Bookmarks Clears all bookmarks in the current script.
Set/Remove Paperclip Creates or removes a paperclip at the cursor in the current script.
Next Paperclip Jumps to the next paperclip in the current script.
Previous Paperclip Jumps to the previous paperclip in the current script.
Clear All Paperclips Clears all paperclips in the current script.
