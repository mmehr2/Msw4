Edit menu commands

The Edit menu offers the following commands:

Undo Reverse previous editing operation.
Cut Deletes data from the document and moves it to the clipboard.
Copy Copies data from the document to the clipboard.
Paste Pastes data from the clipboard into the document.
Paste Special Pastes data from the clipboard into the document in a user-specified format.
Select All Selects all text in the document, for quick copy, font format, or paragraph format.
Find and Replace Searches and optionally replaces characters or words.
Find Next Finds the next occurrence of the last characters specified in the Find and Replace dialog box
