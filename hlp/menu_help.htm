Help menu commands

The Help menu offers the following commands, which provide you assistance with this application:

Help Topics Offers you an index to topics on which you can get help.
Magic Teleprompting on the Web Jumps to the Magic Teleprompting web site where you can find documentation, support, news, and products for your prompting needs.
Register MagicScroll Allows you to enter the registration number for your copy of MagicScroll.
About MagicScroll Displays the version number of this application.
