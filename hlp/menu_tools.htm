Tool menu commands

The Tools menu offers the following commands:

Timer to ART This command sends the current timer value to the ART of the currently active (foreground) script.
Start Timer in Scroll Mode When checked, this option causes the ART timer to start immediately when entering scroll mode.
Reset Timer Sets the ART timer to 0.
Check Spelling Displays a dialog for checking the spelling of the current script or selected text. *For further help, see below.
Speller Options Displays a dialog box for specifying spell checking options. *For further help, see below.
Auto Correct Options Displays a dialog box for specifying auto-correct options.  *For further help, see below.
Got To Line Allows you to move to a specific line in the current script.
Preferences Displays the preferences dialog box

* For further help with spell checking and auto correction, use What's This help. What's This help can be used by clicking on the question mark next to the close button in the top-right corner of a dialog box. After clicking on it, the mouse cursor will change to a question mark icon, which will display help for any control you click on within the dialog box. For example, to understand what the Change button in the Spelling dialog box does, click the question mark in the top right corner of the dialog box, then click on the Change button.
