Preferences

The preferences dialog box allows you to specify most of the configurable setting in MagicScroll. It is available under the Tools menu and provides options for General, Scroll Control, and Video areas.

General preferences include the following settings:

Clear Paperclips in Scroll Mode When transitioning from edit to scroll mode, paperclips are usually retained. Check this option to clear them.
Reset Timer in Scroll Mode When transitioning from edit to scroll mode, the timer is resumed from the last session. Check this option to set the time to 0 each time.
Include Cue bar in Printouts When checked, the cue bar will be printed.
Inverse Video in Edit Mode This option causes white text on a black background to be displayed in scroll mode. Otherwise the scheme is black text on a white background.
ERT Time base The number of characters per second used to calculate the ERT for scripts in the script queue.
Default Font The font to be used when a text (.TXT) file is opened.

Scroll Control preferences include the following settings:

Left Button Choose which action should be taken when the left button is pressed on the mouse or hand controller.
Right Button Choose which action should be taken when the right button is pressed on the mouse or hand controller.
Reverse Sense of Speed Control This option determines how mouse or hand controller movement corresponds with script scrolling.

Video preferences include the following settings:

Color Output When checked, colored text will remain colored in scroll mode. Otherwise, black or white will be used.
Inverse Video When checked, white text will be displayed on a black background in scroll mode.
Script Dividers When checked, a horizontal line will be drawn between scripts in scroll mode.
Show Speed When checked, the speed indicator will be drawn in scroll mode.
Show Script Position When checked, the script position indicator will be drawn in scroll mode.
Show Timer When checked, the ART timer will be drawn in scroll mode.
Loop Scripts When checked, after scrolling beyond the top of the first script or bottom of the last script, MagicScroll will loop to the end of the last or beginning of the first script, respectively.
Link Multiple Scripts When checked, all scripts that are queued in the script queue will be displayed in scroll mode.
Draw Right to Left (Mirror Image) If checked, text will be displayed right to left, or mirrored, in scroll mode.
