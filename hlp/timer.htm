Timer

The timer displays minutes and seconds (mm:ss) in the upper left corner of the scroll window. Its value can be copied to your script's ART (actual run time) value when you return to edit mode.

Press Ctrl+Shift+T to stop or resume the timer. Ctrl+Alt+T can be used to reset the timer to 0.

The timer can be shown or hidden by checking the 'Show Timer' check box on the Video tab of the Preferences dialog box.
