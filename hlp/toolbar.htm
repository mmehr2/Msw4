Toolbar

  • New
 Create a new file.
  • Open
 Open an existing file.
  • Save
 Save the current script.
  • Print
 Print the current script.
  • Print Preview
 Preview the current script.
  • Cut
 Cut the selected text to the clipboard.
  • Copy
 Copy the selected text to the clipboard
  • Paste
 Paste the selected text to the clipboard.
  • Undo
 Undo the last edit.
  • Find and Replace
 Find and optionally replace specified text.
  • Help
 View help contents.

Show or hide the toolbar using the Toolbar command on the View menu.
